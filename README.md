
##Учебный проект по дисциплине 
**«Базы данных»**

## Используемая СУБД

- MySQL ver.- 8.4
- Среда выполнения: OpenServer v.6 / PHPMyAdmin

## Содержание

Репозиторий содержит 4 директории с 4 учебными базами данных
В каждой директории имеетются скрипты для создания таблиц и наполнения их данными (create.sql и insert.sql).
Далее файлы решения задач (Task1.sql, Task2.sql, ...).
